# lichong-resume-online

李冲的在线静态个人简介网站，面向招聘转化展示 AI Agent、全栈交付、前沿技术热爱、DevOps、低代码和可观测性平台经验。

## Commands

- `npm install` 安装依赖
- `npm run dev` 本地开发
- `npm run typecheck` TypeScript 检查
- `npm run lint` 代码规范检查
- `npm run test` 单元/组件测试
- `npm run build` 静态构建
- `npm run test:e2e` Playwright 端到端验证

## Deploy

推送到 `main` 后，GitHub Actions 会构建 `dist` 并发布到 GitHub Pages。
